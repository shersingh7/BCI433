1.0 Introduktion

I den här guiden finns anvisningar för hur du driftsätter produkten på en lokal disk för en enskild användare.

Det finns olika sätt att driftsätta produkten, t.ex. följande:

  • En delad plats som kan nås av flera användare.
    Det finns mer information i IBM i Access Client Solutions: Enkel anpassning och driftsättning
  • Packa upp zip-filen på valfri plats inklusive flyttbara medier (t.ex CD-, DVD- och USB-enhet) eller i en lokal sökväg. Det finns mer information i avsnittet Installation i dokumentet GettingStarted i mappen Documentation.
  • Se Andra driftsättningsalternativ i GettingStarted. Där finns information om alternativ som Importera konfigurationsinställningar automatiskt för spridning av samma konfiguration till flera användare samt IBM i-driftsättningar för direkt användning av kommandoradstilläggsprogram i IBM i.

Driftsättning på lokal disk för en enskild användare beskrivs i resten av det här dokumentet.

2.0 Installation för Windows

Gå till produktens ZIP-fil eller CD.

Om du använder ZIP-filen extraherar du innehållet.
Förslag: Du kan göra det genom att markera filen i Windows Utforskaren, högerklicka på filen och sedan välja Extrahera alla.

Gå till produktens Windows_Application-mapp i Utforskaren.

Installera 32-bitarsversioner för aktuell användare genom att dubbelklicka på install_acs_32.

Installera 64-bitarsversioner för aktuell användare genom att dubbelklicka på install_acs_64.

Installera 32-bitarsversionen för alla användare på en dator genom att dubbelklicka på install_acs_32_allusers.

Installera 642-bitarsversionen för alla användare på en dator genom att dubbelklicka på install_acs_64_allusers.

Om det är en ny installation öppnas flera fönster till hjälp att anpassa den första installationen. Acceptera licensavtalet om det visas.

Om det är en uppgradering behålls tidigare inställningar.

När installationen är klar visas ett slutförandemeddelande.

Vid en installation för en enskild användare kopieras programmet till följande sökväg: 

    /Users/user_id/IBM/ClientSolutions/

Anm. Den första versionen av dessa skript kopierade produktfilerna till en sökväg relativ till MyDocuments. Den aktuella versionen kopierar produktfilerna till en sökväg baserat på miljövariablerna %HOMEDRIVE% och %HOMEPATH%. När skripten används för produktuppdatering sker uppdateringen på den första plats som hittas.

Vid en installation för flera användare kopieras programmet till följande sökväg: 

    /Users/Public/IBM/ClientSolutions/

2.1 Starta produkten i Windows

Dubbelklicka på "Access Client Solutions"-ikonen på skrivbordet.

3.0 Installation för Mac

Gå till produktens ZIP-fil eller CD med Finder.

Om du använder ZIP-filen extraherar du innehållet.
Förslag: Om du precis har hämtat produkten finns den i Downloads. Om innehållet i ZIP-filen inte har extraherats kan du göra det genom att klicka på filen.

Gå till mappen Mac_Application. Dubbelklicka på install_acs.

Om det är en ny installation öppnas flera fönster till hjälp att anpassa den första installationen. Acceptera licensavtalet om det visas.

Om det är en uppgradering behålls tidigare inställningar.

När installationen är klar visas ett slutförandemeddelande.

Programmet kopieras till mappen Applications.

Om du startade installationen från Finder öppnades ett terminalfönster. Du kan stänga terminalfönstret när installationen är klar.

Teknisk anmärkning:
Beroende på säkerhetsinställningarna och vilken OS X-version som används kan ett meddelande visas om att install_acs inte har behörighet att köras. Om det händer måste du ge install_acs behörighet att köra på något av följande sätt:

  • Om du får ett alternativ att öppna install_acs väljer du det.
  • Högerklicka på install_acs och välj Open.
    Välj Open när ett meddelande visas.
  • Om inget av ovanstående fungerar väljer du:
    System Preferences
    Security & Privacy
    Om det finns ett alternativ för att öppna "IBM i Access Client Solutions" markerar du det. I annat fall måste du ändra inställningarna på sidan Allmänt så att andra program får köras. När "IBM i Access Client Solutions" har startats första gången kan du återställa inställningarna.

3.1 Starta produkten på Mac

Använd Finder för att gå till Applications.

Gå till "IBM i Access Client Solutions" och dubbelklicka så startar produkten.

Teknisk anmärkning:
Beroende på säkerhetsinställningarna och vilken OS X-version som används kan ett meddelande visas om att programmet inte har behörighet att köras. Om det händer måste du ge programmet behörighet att köra på något av följande sätt:

  • Om du får ett alternativ att öppna det väljer du det.
  • Högerklicka på ikonen "IBM i Access Client Solutions" och välj Open.
    Välj Open när ett meddelande visas.
  • Om inget av ovanstående fungerar väljer du:
    System Preferences
    Security & Privacy
    Om det finns ett alternativ för att öppna "IBM i Access Client Solutions" markerar du det. I annat fall måste du ändra inställningarna på sidan Allmänt så att andra program får köras. När "IBM i Access Client Solutions" har startats första gången kan du återställa inställningarna.
Teknisk anmärkning:
Om du blir ombedd att installera "Java 6 SE runtime" behöver du inte installera Java 6. Installera Java 8 och försök starta produkten igen.

4.0 Installation för Linux

Gå till produktens ZIP-fil eller CD.

Om du använder ZIP-filen extraherar du innehållet.
Förslag: Kör kommandot unzip från ett terminalfönster.

Gå till mappen Linux_Application.

Eftersom du måste ha rotbehörighet för att använda installationsskriptet använder du sudo för att köra skriptet från ett terminalfönster. Exempel:

Kör följande kommando i 32-bitarsmiljö:

    sudo ./install_acs_32
Kör följande kommando i 64-bitarsmiljö:
    sudo ./install_acs_64

Om det är en ny installation öppnas flera fönster till hjälp att anpassa den första installationen. Acceptera licensavtalet om det visas.

Om det är en uppgradering behålls tidigare inställningar.

När installationen är klar visas ett slutförandemeddelande.

Programmet kopieras till:

    /opt/ibm/iAccessClientSolutions/
    /usr/share/applications/

4.1 Starta produkten i Linux

Sök efter "IBM i Access Client Solutions" i programmappen eller med Super-tangenten och skriv sedan "IBM i Access Client Solutions".

5.0 Installera uppdateringar

När en ny version blir tillgänglig kan du följa installationsanvisningarna i det här dokumentet för att tillämpa uppdateringarna. Om du får en fråga om du vill ersätta befintliga filer väljer du Ja för alla.

Om du använder install_acs-skript för Windows, Mac eller Linux uppdateras inte filen AcsConfig.properties vilket gör att alla konfigurationsändringar du har gjort sparas.

6.0 Mer information

Mer information om hur du anpassar produkten eller om du vill ha information om hur du löser problem vid installation eller start av produkten finns i dokumentet GettingStarted i dokumentationsmappen.

7.0 Avinstallera

Om du vill ta bort tillämpningsfiler som kopierats till ovanstående platser genom att installera skript, använder du det plattformsspecifika avinstallationsskriptet som finns i samma mapp som installationsskriptet: 

    Windows_Application/uninstall_acs.js
    Mac_Application/uninstall_acs
    Linux_Application/uninstall_acs
