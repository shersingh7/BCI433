1.0 Introducción

Esta guía proporciona instrucciones para desplegar este producto en una unidad de disco local para un único usuario.

Hay diversas maneras de desplegar este producto, que incluyen:

  • una ubicación compartida a la que pueden acceder varios usuarios.
    Consulte IBM i Access Client Solutions: Customization and deployment made easy
  • desempaquetar el archivo zip en cualquier ubicación de su elección, incluidos los medios portables (por ejemplo CD, DVD, unidad flash USB) o cualquier vía de acceso de la unidad de disco local. Consulte la sección Instalación del documento Guía de inicio ubicado en la carpeta de documentación
  • Consulte Otras opciones de despliegue en la Guía de inicio para obtener información sobre opciones como por ejemplo Importación automática de valores de configuración y propagar la misma configuración para varios usuarios y Despliegues de IBM i nativos para utilizar plug-ins de línea de mandatos directamente en IBM i.

Para desplegar este producto en la unidad de discos local para un solo usuario, consulte las secciones restantes de este documento.

2.0 Instalación para Windows

Localice el archivo zip o CD del producto.

Si utiliza el archivo zip del producto, extraiga el contenido del archivo zip.
Sugerencia: puede utilizar Windows Explorer para seleccionar el archivo zip, luego pulsar con el botón derecho en el archivo y seleccionar "Extraer todo...".

Mediante Windows Explorer, localice la carpeta Windows_Application del producto.

Para instalar la versión de 32 bits para el usuario actual, haga una doble pulsación en install_acs_32.

Para instalar la versión de 64 bits para el usuario actual, haga una doble pulsación en install_acs_64.

Para instalar la versión de 32 bits para todos los usuarios en un PC de varios usuarios, haga una doble pulsación en install_acs_32_allusers.

Para instalar la versión de 64 bits para todos los usuarios en un PC de varios usuarios, haga una doble pulsación en install_acs_64_allusers.

Si se trata de una instalación nueva, habrá diversas solicitudes que le ayudarán a personalizar la instalación inicial. Si aparece un acuerdo de licencia, acéptelo.

Si se trata de una actualización, se conservarán los valores anteriores.

Una vez finalizada la instalación, se visualizará un mensaje de finalización.

En una instalación satisfactoria de un solo usuario se copiará la aplicación en: 

    /Users/user_id/IBM/ClientSolutions/

Nota: la versión inicial de estos scripts copiaba los archivos de producto en una vía de acceso relativa a MyDocuments. La versión actual copia los archivos de producto en una vía de acceso basada en las variables de entorno %HOMEDRIVE% y %HOMEPATH%. Al utilizar estos scripts para actualizar el producto, la actualización se producirá en la primera ubicación encontrada.

En una instalación satisfactoria de varios usuarios se copiará la aplicación en: 

    /Users/Public/IBM/ClientSolutions/

2.1 Inicio del producto en Windows

Haga una doble pulsación en el icono de "Access Client Solutions" en el escritorio.

3.0 Instalación para Mac

Localice el archivo zip o CD del producto mediante el Buscador.

Si utiliza el archivo zip del producto, extraiga el contenido del archivo zip.
Sugerencia: si solamente ha descargado el producto, busque en Descargas. Si el contenido del archivo zip no se ha extraído, puede extraer el contenido haciendo una doble pulsación en el archivo.

Localice la carpeta Mac_Application. Efectúe una doble pulsación en install_acs.

Si se trata de una instalación nueva, habrá diversas solicitudes que le ayudarán a personalizar la instalación inicial. Si aparece un acuerdo de licencia, acéptelo.

Si se trata de una actualización, se conservarán los valores anteriores.

Una vez finalizada la instalación, se visualizará un mensaje de finalización.

En una instalación satisfactoria se copiará la aplicación en la carpeta Applications.

Si ha iniciado la instalación desde Buscador, se habrá abierto una ventana de terminal para realizar la instalación. Cuando la instalación haya finalizado, puede cerrar la ventana de terminal.

Nota técnica:
En función de los valores de seguridad y de la versión de OS X que esté utilizando, puede que visualice una ventana emergente que indica que install_acs no puede ejecutarse. Si esto sucede, deberá autorizar la ejecución de install_acs mediante uno de los métodos siguientes:

  • Si se presenta una opción para "Abrir" install_acs, selecciónela.
  • Pulse install_acs con el botón derecho del ratón y seleccione "Abrir".
    Si se muestra una ventana emergente, seleccione "Abrir".
  • Si ninguna de las opciones anteriores funciona, vaya a:
    Preferencia del sistema...
    Seguridad & Privacidad
    Si ha una opción para abrir "IBM i Access Client Solutions", selecciónela. De lo contrario, tendrá que ajustar los valores en la pestaña General para permitir que otras aplicaciones se ejecuten. Una vez iniciado "IBM i Access Client Solutions" por primera vez, puede volver a cambiar los valores de seguridad.

3.1 Inicio del producto en Mac

Utilice el Buscador para ir a Aplicaciones.

Localice "IBM i Access Client Solutions" y efectúe una doble pulsación sobre él para iniciar el producto.

Nota técnica:
En función de los valores de seguridad y de la versión de OS X que esté utilizando, puede que visualice una ventana emergente que indica que la aplicación no puede ejecutarse. Si esto sucede, deberá autorizar la ejecución de la aplicación mediante uno de los métodos siguientes:

  • Si se presenta una opción para "Abrir", selecciónela.
  • Pulse el icono "IBM i Access Client Solutions" con el botón derecho del ratón y seleccione "Abrir".
    Si se muestra una ventana emergente, seleccione "Abrir".
  • Si ninguna de las opciones anteriores funciona, vaya a:
    Preferencia del sistema...
    Seguridad & Privacidad
    Si ha una opción para abrir "IBM i Access Client Solutions", selecciónela. De lo contrario, tendrá que ajustar los valores en la pestaña General para permitir que otras aplicaciones se ejecuten. Una vez iniciado "IBM i Access Client Solutions" por primera vez, puede volver a cambiar los valores de seguridad.
Nota técnica:
Si se le solicita que instale "Java 6 SE runtime", no necesita instalar Java 6. Instale Java 8 y luego intente volver a iniciar el producto.

4.0 Instalación para Linux

Localice el archivo zip o CD del producto.

Si utiliza el archivo zip del producto, extraiga el contenido del archivo zip.
Sugerencia: desde una ventana de terminal, utilice el mandato unzip.

Localice la carpeta Linux_Application.

Puesto que deberá tener autorización root para utilizar el script de instalación, utilice sudo para ejecutar el script de instalación desde una ventana de terminal. Por ejemplo:

Si está en una arquitectura de 32 bits, utilice el mandato siguiente: 

    sudo ./install_acs_32
Si está en una arquitectura de 64 bits, utilice el mandato siguiente: 
    sudo ./install_acs_64

Si se trata de una instalación nueva, habrá diversas solicitudes que le ayudarán a personalizar la instalación inicial. Si aparece un acuerdo de licencia, acéptelo.

Si se trata de una actualización, se conservarán los valores anteriores.

Una vez finalizada la instalación, se visualizará un mensaje de finalización.

En una instalación satisfactoria se copiará la aplicación en:

    /opt/ibm/iAccessClientSolutions/
    /usr/share/applications/

4.1 Inicio del producto en Linux

Busque "IBM i Access Client Solutions" en la carpeta de aplicaciones o utilizando la tecla Super y luego escribiendo "IBM i Access Client Solutions".

5.0 Instalación de actualizaciones

Cuando haya una versión nueva disponible, puede seguir las instrucciones de instalación de este documento para aplicar las actualizaciones. Si se le pregunta si desea sobrescribir los archivos existentes, responda "Sí a todo".

Si utiliza los scripts install_acs de Windows, Mac o Linux, el archivo AcsConfig.properties no se actualizará, por lo que los cambios de configuración personalizados que haya realizado en ese archivo no se perderán.

6.0 Más información

Para obtener más información sobre la personalización del producto o si experimenta problemas al instalar o iniciar el producto, consulte el documento Guía de inicio ubicado en la carpeta Documentación.

7.0 Desinstalar

Para eliminar los archivos de aplicación copiados en las ubicaciones anteriores por el script de instalación, utilice el script de desinstalación que se encuentra en la misma carpeta que el script de instalación: 

    Windows_Application/uninstall_acs.js
    Mac_Application/uninstall_acs
    Linux_Application/uninstall_acs
