1.0 Введение

В данном руководстве приведены инструкции по развертыванию продукта на локальном жестком диске для одного пользователя.

Существует ряд других способов развернуть этот продукт:

  • общее расположение, доступное многим пользователям.
    См. документ IBM i Access Client Solutions: Customization and deployment made easy
  • распаковка архива zip в любом расположении, включая съемный носитель (компакт-диск, диск DVD, флэш-накопитель USB), или в любом каталоге на локальном диске. См. раздел Установка документа GettingStarted, расположенного в папке Documentation
  • В разделе Другие способы развертывания документа GettingStarted описаны такие варианты, как Автоматический импорт параметров конфигурации для настройки одинаковой конфигурации у нескольких пользователей и Развертывание напрямую в IBM i для применения модулей команд непосредственно в IBM i.

Для того чтобы развернуть этот продукт на локальном диске для одного пользователя, ознакомьтесь с приведенной ниже информацией.

2.0 Установка в Windows

Найдите архивный файл zip или компакт-диск продукта.

При наличии архивного файла zip распакуйте его.
Рекомендация: для этого можно выбрать архивный файл zip в Проводнике Windows, щелкнуть на нем правой кнопкой мыши и выбрать команду "Извлечь все...".

Перейдите в папку Windows_Application продукта в Проводнике Windows.

Для того чтобы установить 32-разрядную версию для текущего пользователя, дважды щелкните на файле install_acs_32.

Для того чтобы установить 64-разрядную версию для текущего пользователя, дважды щелкните на файле install_acs_64.

Для того чтобы установить 32-разрядную версию для всех пользователей ПК, дважды щелкните на файле install_acs_32_allusers.

Для того чтобы установить 64-разрядную версию для всех пользователей ПК, дважды щелкните на файле install_acs_64_allusers.

Если продукт устанавливается впервые, вам будет предложено указать несколько значений для первоначальной настройки продукта. Если будет открыто лицензионное соглашение, примите его.

Если выполняется обновление продукта, будут сохранены текущие параметры.

После завершения установки появится сообщение о результатах выполнения операции.

В случае успешной установки продукта для одного пользователя он будет скопирован в следующий каталог: 

    /Users/ИД-пользователя/IBM/ClientSolutions/

Примечание: в начальной версии этих сценариев файлы продукта копировались в каталог MyDocuments. В текущей версии для копирования файлов продукта используются переменные среды %HOMEDRIVE% и %HOMEPATH%. В случае применения этих сценариев для обновления продукта применяется первое обнаруженное расположение.

В случае успешной установки продукта для нескольких пользователей он будет скопирован в следующий каталог: 

    /Users/Public/IBM/ClientSolutions/

2.1 Запуск продукта в Windows

Дважды щелкните на значке "Access Client Solutions", расположенном на рабочем столе.

3.0 Установка в Mac

Найдите архивный файл zip продукта или компакт-диск продукта с помощью программы Finder.

При наличии архивного файла zip распакуйте его.
Рекомендация: если продукт был только что загружен, посмотрите в папке Загрузки. Если содержимое архива zip еще не распаковано, то дважды щелкните на файле, чтобы распаковать его.

Найдите папку Mac_Application. Дважды щелкните на файле install_acs

Если продукт устанавливается впервые, вам будет предложено указать несколько значений для первоначальной настройки продукта. Если будет открыто лицензионное соглашение, примите его.

Если выполняется обновление продукта, будут сохранены текущие параметры.

После завершения установки появится сообщение о результатах выполнения операции.

В случае успешной установки приложение будет скопировано в папку Приложения.

Если установка была запущена из Finder, для ее выполнения будет открыто окно терминала. После завершения установки окно терминала можно закрыть.

Техническое примечание:
В ряде версий OS X при определенных настройках защиты может появится всплывающее окно с информацией о том, что запуск файла install_acs запрещен. В этом случае необходимо разрешить запуск install_acs одним из следующих способов:

  • Если предусмотрена команда "Открыть" install_acs, выберите ее.
  • Щелкните правой кнопкой мыши на файле install_acs и выберите "Открыть".
    Когда появится всплывающее окно, выберите "Открыть".
  • Если ни один из указанных выше способов не работает, то выберите:
    Системные настройки...
    Защита и безопасность
    Если будет доступна команда Открыть "IBM i Access Client Solutions", выберите ее. В противном случае вам потребуется изменить параметры на вкладке Общие, разрешив запуск других приложений. После первого запуска "IBM i Access Client Solutions" можно восстановить прежние настройки безопасности.

3.1 Запуск продукта в Mac

Перейдите в папку Программы с помощью Finder.

Найдите "IBM i Access Client Solutions" и дважды щелкните на этой программе для ее запуска.

Техническое примечание:
В ряде версий OS X при определенных настройках защиты может появится всплывающее окно с информацией о том, что запуск приложения запрещен. В этом случае необходимо разрешить запуск приложения одним из следующих способов:

  • Если предусмотрена команда "Открыть" приложение, выберите ее.
  • Щелкните правой кнопкой мыши на значке "IBM i Access Client Solutions" и выберите "Открыть".
    Когда появится всплывающее окно, выберите "Открыть".
  • Если ни один из указанных выше способов не работает, то выберите:
    Системные настройки...
    Защита и безопасность
    Если будет доступна команда Открыть "IBM i Access Client Solutions", выберите ее. В противном случае вам потребуется изменить параметры на вкладке Общие, разрешив запуск других приложений. После первого запуска "IBM i Access Client Solutions" можно восстановить прежние настройки безопасности.
Техническое примечание:
Если вам было предложено установить "Java 6 SE runtime", то не нужно устанавливать Java 6. Установите Java 8 и попробуйте снова запустить продукт.

4.0 Установка в Linux

Найдите архивный файл zip или компакт-диск продукта.

При наличии архивного файла zip распакуйте его.
Рекомендация: используйте команду unzip в окне терминала.

Найдите папку Linux_Application.

Поскольку сценарий установки разрешено запускать только пользователю root, для запуска сценария установки из окна терминала используйте команду sudo. Пример:

На компьютере с 32-разрядной архитектурой введите следующую команду:

    sudo ./install_acs_32
На компьютере с 64-разрядной архитектурой введите следующую команду:
    sudo ./install_acs_64

Если продукт устанавливается впервые, вам будет предложено указать несколько значений для первоначальной настройки продукта. Если будет открыто лицензионное соглашение, примите его.

Если выполняется обновление продукта, будут сохранены текущие параметры.

После завершения установки появится сообщение о результатах выполнения операции.

В случае успешной установки файлы приложения будут скопированы в следующий каталог:

    /opt/ibm/iAccessClientSolutions/
    /usr/share/applications/

4.1 Запуск продукта в Linux

Найдите "IBM i Access Client Solutions" в папке приложений или нажмите клавишу Super и введите "IBM i Access Client Solutions".

5.0 Установка обновлений

После выхода обновления выполните приведенные здесь инструкции по установке для применения обновления. Когда будет предложено заменить существующие файлы, выберите вариант ответа "Да для всех".

При использовании сценариев install_acs в Windows, Mac или Linux файл AcsConfig.properties не обновляется, поэтому все внесенные в него изменения будут утеряны.

6.0 Дополнительные сведения

Дополнительные сведения о настройке продукта и устранении неполадок, возникающих при установке или запуске продукта, можно найти в документе GettingStarted, расположенном в папке Documentation.

7.0 Удаление продукта

Для удаления файлов приложений, скопированных в указанные выше каталоги сценарием установки, вызовите сценарий удаления для своей платформы, расположенный в той же папке, что и сценарий установки: 

    Windows_Application/uninstall_acs.js
    Mac_Application/uninstall_acs
    Linux_Application/uninstall_acs
